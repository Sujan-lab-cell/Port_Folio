# Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, experience, and achievements as an AI/ML and Software Developer.

## Live Demo

Portfolio: https://port-folio-sujan.vercel.app/

## Features

* Responsive design for desktop, tablet, and mobile devices
* Professional landing page
* About Me section
* Skills showcase
* Project showcase with GitHub links
* Contact section
* Smooth user experience
* Fast performance with Vite

## Installation
``` text
Clone the repository:

git clone https://github.com/Sujan-lab-cell/Port_Folio.git

Navigate to the project directory:

cd your-repository

Install dependencies:

npm install

Start the development server:

npm run dev

Build for production:

npm run build
```
## Deployment

The project is deployed using Vercel.
